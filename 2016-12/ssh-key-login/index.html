plink

plink.exe使用private key 時，必須把通過ssh生成的private key通過 PuTTYgen.exe轉換一下

  • 打開 puttygen
  • 選擇 Conversions->Import key 導入 private key
  • 點擊按鈕 Save private key

scp

使用private key 登錄。

  • -i ~/.ssh/as_rsa 對應的private key
