R1soft 備份失敗

发表于 2017-02-24 更新于 2018-11-12

安裝錯誤

錯誤訊息提示要執行r1soft-setup --get-module.
但是報錯：kernel的版本信息和kernel-devel 版本對不上。

所以一次執行：

  • yum install kernel-devel-$(uname -r)
  • r1soft-setup –get-module –no-binary
  • /etc/init.d/cdp-agent restart
