golang package

发表于 2019-08-03 更新于 2019-08-11

定义

go import 的包是由 包路径/包名 组合而成的。

  • 目录名称必须和包的名称一样
  • 统一个目录下只能定义一个包名

import 别名

import 特殊

点操作

. 导入可以让包内的方法注册到当前包的上下文中，直接调用方法名即可，不需要再加包前缀。

下划线操作

_ 是包引用操作，只会执行包下各模块中的 init 方法，并不会真正的导入包，所以不可以调用包内的其他方法。

每个包中可以在任意文档中定义 init 方法。 如果定义多个，则都会加载。
