sar 常用命令 & 查看系統流量

发表于 2017-12-01 更新于 2019-03-25

sar -n DEV 2 4

显示网络接口信息 命令后面2 4 意思是：每2秒钟取1次值，取4次。

  • IFACE：LAN接口
  • rxpck/s:每秒钟收到数据包的数量.
  • txpck/s:每秒钟发送数据包的数量.
  • rxkB/s:每秒钟接收的字节(KB).
  • txkB/s:每秒钟发送的字节(KB).
  • rxcmp/s:每秒收到的压缩包的数量
  • txcmp/s:每秒发出的压缩包的数量
  • rxmcst/s:每秒收到的广播包的数量

常用命令

  • sar 5 5 // CPU和IOWAIT统计状态
  • sar -p 1 3 // cpu 狀態
  • sar -n DEV 2 4 // 網絡
