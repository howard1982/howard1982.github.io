rsync 使用說明

发表于 2021-06-21

安裝

yum -y install rsync

使用說明

rsync -av /var/www/html /backup/2020-05-24.0331/web

參數說明

  • -v, –verbose 详细模式输出
  • -a, –archive 归档模式，表示以递归方式传输文件，并保持所有文件属性，等于-rlptgoD
  • --delete 删除那些DST中SRC没有的文件
  • --exclude 排除
  • --include 包含，

例如 rsync -av --include="*.txt" --exclude='*' source/ destination

  • --link-dest 增量備份
