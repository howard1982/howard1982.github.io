mysql sql-mode

发表于 2018-10-30

Mysql執行Sql語句時出現不兼容的問題一般情況下是 默認的 sql-mode 不一樣了。

只要修改sql-mode就可以了。

/etc/my.cnf 中增加 sql-mode="NO_AUTO_CREATE_USER,NO_ENGINE_SUBSTITUTION"

詳見 mysqld_sql-mode
