log

常用命令

查看邮件队列

  • mailq
  • postqueue -p

    查看內容

  • postcat -q DBA3F1A9

其他

  • postsuper -d DBA3F1A9删除队列里的邮件
  • postsuper -d ALL删除队列里所有的邮件
  • postsuper -r ALL重新排队所有邮件
