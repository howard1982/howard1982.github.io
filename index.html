Apache MPM

发表于 2021-06-08

prefork MPM

Implements a non-threaded, pre-forking web server
prefork模式可以算是很古老但是非常稳定的Apache模式。

  • 缺点：占用内存多。不擅长处理高并发请求。

worker MPM

Multi-Processing Module implementing a hybrid multi-threaded multi-process web server
worker模式使用了多进程和多线程的混合模式

-优点：占据更少的内存，高并发下表现更优秀。

event MPM

A variant of the worker MPM with the goal of consuming threads only for connections with active processing
最新的模式，worker的一个变体。它和worker模式很像。最大的区别在于，它解决了keep-alive 场景下，长期被占用的线程的资源浪费问题

选择

要求 更高伸缩性的站点可以选择使用线程的 MPM，即 worker 或 event； 需要可靠性或者与旧软件兼容的站点可以使用 prefork。

Jquery prop vs attr

发表于 2017-07-07 更新于 2021-02-05

如果是用於判斷 盡量使用 $(elem).prop("checked")。 適用於 checked, selected, or disabled

使用 $(elem).prop("checked", "true" ) 來設定 是否點選

elem.checked

true (Boolean) Will change with checkbox state

$(elem).prop(“checked”)

true (Boolean) Will change with checkbox state

elem.getAttribute(“checked”)

"checked" (String) Initial state of the checkbox; does not change

$(elem).attr(“checked”) (1.6.1+)

"checked" (String) Will change with checkbox state

$(elem).attr(“checked”) (pre-1.6)

true (Boolean) Changed with checkbox state
