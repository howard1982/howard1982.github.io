docker 初探

发表于 2020-04-21

基本操作

執行一個文檔

docker run -v /root/myapp:/usr/src/myapp -w /usr/src/myapp php php i.php

  1. -v /root/myapp:/usr/src/myapp : 将主机中当前目录下的 myapp 挂载到容器的 /usr/src/myapp。
  2. -w /usr/src/myapp : 指定容器的 /usr/src/myapp 目录为工作目录。
  3. php : image 名稱
  4. php i.php : 使用容器的 php 命令来执行工作目录中的 i.php 文件。

啓動nginx

docker run --name nginx-test -p 8080:80 -d nginx

  1. –name nginx-test：容器名称。
  2. -p 8080:80： 端口进行映射，将本地 8080 端口映射到容器内部的 80 端口。
  3. -d nginx： 设置容器在在后台一直运行。

執行一個命令

  1. docker run php:latest php -v
  2. php -v 要執行的內容
  3. php:latest php image名稱 latest tag名稱

交互式

docker run -it php:latest /bin/bash

容器 container

停止

docker stop <CONTAINER ID>

删除

docker rm <CONTAINER ID>

啓動

docker start <CONTAINER ID>
