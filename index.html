sar 常用命令 & 查看系統流量

发表于 2017-12-01 更新于 2021-06-21

安裝

yum install sysstat

說明

sar -n DEV 2 4

显示网络接口信息 命令后面2 4 意思是：每2秒钟取1次值，取4次。

  • IFACE：LAN接口
  • rxpck/s:每秒钟收到数据包的数量.
  • txpck/s:每秒钟发送数据包的数量.
  • rxkB/s:每秒钟接收的字节(KB).
  • txkB/s:每秒钟发送的字节(KB).
  • rxcmp/s:每秒收到的压缩包的数量
  • txcmp/s:每秒发出的压缩包的数量
  • rxmcst/s:每秒收到的广播包的数量

常用命令

  • sar 5 5 // CPU和IOWAIT统计状态
  • sar -p 1 3 // cpu 狀態
  • sar -n DEV 2 4 // 網絡

Apache MPM

发表于 2021-06-08

prefork MPM

Implements a non-threaded, pre-forking web server
prefork模式可以算是很古老但是非常稳定的Apache模式。

  • 缺点：占用内存多。不擅长处理高并发请求。

worker MPM

Multi-Processing Module implementing a hybrid multi-threaded multi-process web server
worker模式使用了多进程和多线程的混合模式

-优点：占据更少的内存，高并发下表现更优秀。

event MPM

A variant of the worker MPM with the goal of consuming threads only for connections with active processing
最新的模式，worker的一个变体。它和worker模式很像。最大的区别在于，它解决了keep-alive 场景下，长期被占用的线程的资源浪费问题

选择

要求 更高伸缩性的站点可以选择使用线程的 MPM，即 worker 或 event； 需要可靠性或者与旧软件兼容的站点可以使用 prefork。

php 錯誤處理

发表于 2017-12-05 更新于 2021-05-09

打印錯誤

直接在頁面輸出錯誤報告

  • display_errors 是否打開顯示錯誤報告
  • error_reporting 輸出錯誤等級

  • log_errors 是否存儲
